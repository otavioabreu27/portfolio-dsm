Apresentação

Olá, sou o Otávio, um desenvolvedor de software com foco em backend com experiência na construção de soluções escaláveis e arquiteturas robustas. Minha atuação envolve a modernização de aplicações legadas, otimização de desempenho e desenvolvimento de APIs eficientes para análise e processamento de dados. Possuo habilidades em diversas tecnologias, incluindo Python, Java, Rust, e frameworks como FastAPI, Spring Boot e NestJS. Tenho experiência em bancos de dados relacionais e NoSQL, além de práticas DevOps, como Docker, Swarm e CI/CD. Busco constantemente aprimorar minha expertise para entregar soluções de alto valor.

Projetos

ABP 1º Semestre - Py.nk

Sistema de internet banking com funções básicas de funcionamento e uma interface de administração para gerenciamento da aplicação. Permite o cadastro e login de usuários, a realização de consultas de extratos, movimentações financeiras como depósitos, saques e transferências, além da geração de comprovantes para cada transação. O sistema também possibilita solicitações de atualização de dados e fechamento de contas, além de contar com um painel administrativo para aprovação ou rejeição de solicitações, conferência de depósitos e gerenciamento de agências e gerentes. Além disso, o sistema implementa mecanismos de controle para arredondamento de valores e cálculo de rendimento de poupança.

Python Flask MySQL HTML CSS

Repositório no GitHub

ABP 2º Semestre - TSwift

Sistema para controle de recebimento de grãos no setor agroindustrial. Permite o gerenciamento da entrada, inspeção e acompanhamento em tempo real das mercadorias recebidas. Conta com três perfis de usuários (administrativo, operacional e gerencial) e funcionalidades para cadastro de usuários, produtos e transportadoras, além de controle de acesso e geração de relatórios gerenciais.

Typescript Prisma React Bootstrap MySQL

Repositório no GitHub

ABP 3º Semestre - TerraGeo Manager

Plataforma para gestão e monitoramento de áreas agrícolas, com funcionalidades estatísticas para acompanhamento da produtividade e administração. Permite atribuir e classificar quadrículas, oferecendo uma visão detalhada do desempenho da produção.

Typescript MongoDB React MUI

Repositório no GitHub

ABP 4º Semestre - Nutrilife

O Nutrilife é um aplicativo de controle nutricional. Ele permite que usuários preencham uma anamnese e calculem seu metabolismo basal. Além disso, o sistema oferece funcionalidades como monitoramento de progresso, substituição de alimentos na dieta e integração com dispositivos inteligentes para um acompanhamento detalhado da nutrição.

Typescript PostgreSQL DigitalOcean Github Actions ReactNative
